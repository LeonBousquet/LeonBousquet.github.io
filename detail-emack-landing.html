Description

EMACK est le nom de l'application web dont je suis co-créateur en tant qu'auto-entrepreneur avec deux amis de mon école. Il s'agit d'un application destinée aux enseignants en informatique dans le supérieur. Elle propose des mini jeux permettant de pratiquer des notions d'informatique telles que le Python, les commandes Linux ou encore le SQL de façon ludique. Pour ce faire un terminal Linux et un IDE sont directement intégrés dans l'application. Pour en apprendre plus sur EMACK, consultez le lien de la landing page juste au-dessus.
Technologies utilisées : PHP (framework Codeigniter), HTML/CSS, Javascript, JQuery, Python, Docker, SQL
